🐙 Deploy — GitHub Pages
Hospede gratuitamente no GitHub

Siga os passos abaixo para publicar o dashboard online:

1. Criar repositório
# No GitHub, crie um repositório público
# Salve este arquivo como index.html
2. Subir o arquivo
git init
git add index.html
git commit -m "feat: qa dashboard"
git branch -M main
git remote add origin https://github.com/SEU_USER/REPO.git
git push -u origin main
3. Ativar GitHub Pages
Repositório → Settings → Pages
Source: Deploy from branch
Branch: main / (root)

✓ URL: https://SEU_USER.github.io/REPO/
4. (Opcional) Domínio customizado
Settings → Pages → Custom domain
Ex: qa.rochlinging.com.br
